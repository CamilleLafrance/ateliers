Ecrire une fonction qui reçoive une chaine de 0 à 100 caractères contenant des parenthèses et qui sache dire si leur ordre est valide,  en retournant true ou false dans le cas échéant : 
````
"()"              =>  true
"("               =>  false
"()()"            =>  true
"())(()"          =>  false
")(()))"          =>  false
"(())((()())())"  =>  true
````
